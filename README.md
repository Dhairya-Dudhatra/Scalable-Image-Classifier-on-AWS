# Scalable-Image-Classifier-on-AWS
This is my mini academic project in CSE546.

This branch is for extra work that I am doing to learn a few AWS services like **AWS Cloud Formation** and **AWS Code Pipeline**.

AWS CloudFormation:
I am creating a stack which can create a web-tier instance, two SQS queues, two S3 buckets, two CloudWatch alarms and AWS Autoscaler for apptier instances.

AWS Code Pipeline:
This service will be mainly use to automatically build, test and deploy the code into webtier and apptier instances.
